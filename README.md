Это полезный инструмент, который поможет найти страницу жертвы, если она использует один никнейм в разных социальных сетях.

Установка:
$ apt update
$ apt upgrade
$ apt install git
$ git clone https://github.com/issamelferkh/userrecon.git

Запуск:
$ cd userrecon
$ chmod +x userrecon.sh
$ bash userrecon.sh
